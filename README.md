# Project Amethyst
## Building from source
I'm assuming you already have a terminal open in the project folder and Node.js installed.
1. Run `npm i` to install dependencies
2. Get an `initramfs` folder
3. To build, run `npm run build`
## Getting initramfs
Unzip `default_initramfs.zip` into the project's root directory.
If you've done this correctly, your folder should now have an `initramfs` folder with `init.js` in it
